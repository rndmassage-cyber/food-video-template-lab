# templates/prompt-packs/

Prompt packs для шаблонов.

Здесь будут размещаться наборы промптов, привязанные к конкретным шаблонам: motion prompt blocks, system-style instructions, negative/anti-drift guidance и template-specific overrides.

Формат prompt pack будет определён в Фазе 2 (Template schema и prompt-pack foundation).
