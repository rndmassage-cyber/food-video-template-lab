# templates/specs/

Спецификации шаблонов (template specs).

Здесь будут размещаться формальные описания каждого шаблона: идентификатор, family, допустимые параметры, motion intent, prompt strategy, scoring criteria и статус.

Формат template spec будет определён в Фазе 2 (Template schema и prompt-pack foundation).
